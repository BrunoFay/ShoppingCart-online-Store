# Shopping cart online store [ENGLISH]

Project carried out in a group in the Trybe course.
The goal was to make an e-commerce using the Mercado Livre API.

## Lessons learned:
- Rotas com BrowserRouter,switch, link.
- Prop Drilling 
- Consumo de api
- Metodologia ágil
- Css grid/flexbox
